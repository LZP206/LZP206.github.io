一.信道容量模型

信道容量表达式：

R=B×[log⁡2(1+γ)−VBτQ−1(ε)]  (bits/s)(1.1)R = B \times \left[ \log_2 (1+\gamma) - \sqrt{\frac{V}{B\tau}}Q^{-1}(\varepsilon) \right] {\text{\;}} (\rm{bits/s}) \tag{1.1}R=B×[log2​(1+γ)−BτV​​Q−1(ε)](bits/s)(1.1)

其中，V=[1−(1+γ)−2]⋅(log⁡2e)2V=\left[1-(1+\gamma)^{-2}\right] \cdot\left(\log _{2} e\right)^{2}V=[1−(1+γ)−2]⋅(log2​e)2表示信道色散，且log⁡2e=ln⁡e/ln⁡2=1/ln⁡2\log _{2} e=\ln e / \ln 2=1 / \ln 2log2​e=lne/ln2=1/ln2。因此有变式：

R=Bln⁡2×[ln⁡(1+γ)−1−(1+γ)2B⋅τQ−1(ε)](1.2)R=\frac{B}{\ln 2} \times\left[\ln (1+\gamma)-\sqrt{\frac{1-(1+\gamma)^{2}}{B \cdot \tau}} Q^{-1}(\varepsilon)\right] \tag{1.2}R=ln2B​×[ln(1+γ)−B⋅τ1−(1+γ)2​​Q−1(ε)](1.2)

% Input(4):
% bandwidth —— 带宽
% gamma —— 信噪比SNR
% tau —— 传输时间
% varepsilon —— 误码率

function result = achievable_capacity(bandwidth, gamma, tau, varepsilon)
    channel_dispersion = ( 1 - (1+gamma)^(-2) ) * ( log2( exp(1) ) )^2;
    capacity = bandwidth * ( log2(1+gamma) - sqrt( channel_dispersion / (bandwidth*tau) ) * qfuncinv(varepsilon) );
    result = capacity;
end

二.解码误码率模型

解码误码率表达式：

% 解码误码率表达式1
% Input(4):
% bandwidth -- 带宽
% gamma -- 信噪比SNR
% tao -- 传输时间
% rate -- 码速率（信道容量）
function result = decoding_error_probability1(bandwidth, gamma, tau, rate)
    channel_dispersion = ( 1 - (1+gamma)^(-2) ) * ( log2( exp(1) ) )^2;
    f = sqrt( bandwidth * tau / channel_dispersion ) * ( log2(1+gamma) - (rate/bandwidth) );
    error_probability = qfunc(f);
    result = error_probability;
end

% 解码误码率表达式2
% Input(3):
% blocklength -- 码长
% gamma -- 信噪比SNR
% packet_size -- 短包大小
function result = decoding_error_probability2(blocklength, gamma, packet_size)
    f = log(2) * sqrt( blocklength / ( 1 - (1+gamma)^(-2) ) ) * ( log2(1+gamma) - (packet_size/blocklength) );
    error_probability = qfunc(f);
    result = error_probability;
end
